Computation Camp for PhD Students

Schedule

Week 1: Intro

Lecture 1 – Intro to Programming and Julia

  1. Comparison of different languages

  2. Setting up a Julia environment

  3. Basics – printing, vectors, loops

  4. Using packages

  5. Coding example: white noise plot, simple OLS, optimal growth model

Lecture 2 – Proper usage: maximizing performance and version control

  1. Good coding practices.

  2. Background: explanation of compilation

  3. Optimizing Julia’s performance and Julia’s profiler

  4. Introduction to Git

  5. Coding example: optimized optimal growth model.

Week 2: Advanced Usage

Lecture 3 – Optimization

  1. Introduction to optimization

  2. Coding example: optimizing the Rosenbrock function with Newton’s method, LBFGS, and Nelder-Mead.

  3. Other applications

Lecture 4 – Interpolation

  1. Explanation of the idea
  2. Linear and spline interpolation
  3. Applications

Week 3: Dynamic Programming

Lecture 5 – Intro

  1. Explanation of the idea
  2. Run through classic problems: Fibonacci sequence, egg dropping, etc.

Lecture 6 – Applications to Economics

  1. High-level taxonomy of economic models.
  2. Coding example: backward induction discrete-time Ben-Porath problem.

Week 4: Stochastic Methods

Lecture 7 – Parallelization and Monte-Carlo

  1. Special lecture from Russell Dimond

Lecture 8 – Monte-Carlo

  1. Birthday paradox; wage shocks.
